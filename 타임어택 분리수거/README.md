# 분리수거 미니게임

## 반영 사항

- 제한시간: 30초
- 문제 데이터: 134개
- 난이도: 0~10초 쉬움, 10~20초 보통, 20~30초 어려움
- 오답 처리: 선택지 중앙에 큰 해설을 0.9초 표시한 뒤 자동으로 다음 문제 진행
- 분류: 일반 / 플라스틱·비닐 / 종이·종이팩 / 캔·유리·고철 / 음식물 / 전용 수거
- 지역 차이가 있는 품목은 `지역 확인` 배지 표시

## 기준

환경부의 핵심 원칙인 비우기, 헹구기, 분리하기, 섞지 않기를 기본으로 구성했습니다.
다만 음식물, 깨진 유리, 대형폐기물, 폐의약품, 폐식용유 등은 지역별 수거 방식이 달라
게임의 `regional: true` 항목에는 지자체 기준을 확인하도록 표시했습니다.

## 앱 연동

```js
window.addEventListener("game:finished", (event) => {
  const { score, maxCombo, duration } = event.detail;
});
```

`index.html` 하나만 열면 실행됩니다.
